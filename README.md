## Simple Site for Practicing JWT Token


### Requirement
- python3
- graphene-django
- djongo
- bcrypt
- graphene-federation


### Environment Variable
- MONGODB_HOST
- JWT_SECRET
- JWT_ISSUER
- JWT_AUDIENCE


### Development Server
```
python3 manage.py runserver 8080
```


### Interactive Python Shell
```
python3 manage.py shell
```
